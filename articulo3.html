¿Qué es JavaScript

JavaScript es uno de los lenguajes de programación más populares y esenciales en el desarrollo web. Junto con HTML y CSS, forma la base de la mayoría de los sitios web modernos. Si alguna vez has interactuado con una página web (como hacer clic en un botón, enviar un formulario o ver una animación), probablemente esa acción fue controlada por JavaScript.

Características principales:

  • Lenguaje interpretado: No necesita ser compilado.
  • Basado en objetos: Usa objetos para organizar datos y funcionalidades.
  • Event-driven: Responde a eventos como clics, movimientos del ratón, etc.
